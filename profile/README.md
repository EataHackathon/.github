> ### 1. 서비스 소개
EATAMAMA는 임산부의 **주차, 질환, 알레르기** 등 **개인 건강 정보**를 반영해  
**안전한 음식 선택과 식단 추천**을 제공하는 서비스입니다.

### 이 프로젝트가 해결하는 문제
1.	음식 정보에 대한 검색피로와 불안감 해결
2.	개인 맞춤 식단 정보 부재 해결
3.	요리·재료 단위의 확인 불가 문제 해결

> ### 주요 기능
- **음식 검색**: 식/재료 검색 → 위험/안전 여부 및 AI 요리추천 또는 AI 재료 분석
